Formulir validasi alat ini dapat memeriksa diperlukan, bidang bilangan bulat, email dan tanggal dalam bentuk Anda. Itu portabel untuk semua bentuk Anda dan akan membantu menyelamatkan sisi server sumber daya dari lapangan validasi form dan page ulang. Semua pesan yang disesuaikan dalam formulir tersembunyi elemen di mana Anda juga memilih persyaratan validasi. Anda dapat mencampur elemen yang dibutuhkan dengan salah satu unsur-unsur lain. Kode ini benar-benar portabel dan dapat digunakan dengan bentuk oleh hanya menambahkan elemen tersembunyi Anda tepat dan menambahkan onClick perintah untuk sebuah tombol.

Mengikuti aturan.
1) Untuk mengatur penggunaan diperlukan bidang nama field yang ada dan menambahkan r_ ke awal bidang nama dan membuat elemen tersembunyi.
CONTOH:
<input type="text" name="your_fieldname">
<input type="hidden" name="r_your_fieldname" value="Required error message goes here">

2) Untuk mengatur integer lapangan menggunakan nama field yang ada dan menambahkan i_ ke awal bidang nama dan membuat elemen tersembunyi.
CONTOH:
<input type="text" name="your_fieldname">
<input type="hidden" name="i_your_fieldname" value="Integer error message goes here">

3) Untuk menetapkan bidang tanggal penggunaan nama field yang ada dan menambahkan D_ ke awal bidang nama dan membuat elemen tersembunyi. Ini field tanggal yang hanya menerima MM/DD/YY format atau M/D/YY.
CONTOH:
<input type="text" name="your_fieldname">
<input type="hidden" name="d_your_fieldname" value="Date error message goes here">

4) Untuk memvalidasi email lapangan menggunakan nama field yang ada dan Tambahkan e_ ke awal nama field dan membuat tersembunyi elemen.
CONTOH:
<input type="text" name="your_fieldname">
<input type="hidden" name="e_your_fieldname" value="The email address format is incorrect">

Inisialisasi script.
Anda harus menggunakan jenis = "tombol" bukan tipe = "Kirim" tombol Kirim Anda, javascript mengirim Kirim untuk Anda.
CONTOH:
<input type=button Value="Button Message Here" onclick="check(form,form.elements.length)">
