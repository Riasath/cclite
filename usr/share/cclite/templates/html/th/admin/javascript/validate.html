ผมต้องขออภัยสำหรับความจริงที่ฉัน ไม่สามารถรับอีเมล์ถามเกี่ยวกับสคริปต์ของฉันที่ฉันต้องการ หลายชั่วโมงเพื่อช่วยให้ผู้ใช้ได้อย่างถูกต้องรวมโมดูลนี้เท่าใด และ เขียนรายการเพิ่มเติมสำหรับทุกคน นี้เกิดขึ้นเนื่องจากอีเมล์ หนอนของ 2003 สิงหาคมละเมิดที่อยู่อีเมล์ของฉันแสดงอย่างรุนแรง โจมตีไปที่มีการจัดการกับพันที่มีข้อผิดพลาดต่อไปนี้ ส่งกลับมาที่อีเมล์วัน นอกจากนี้ฉันเป็นระยะ ๆ อาจพยายามที่จะเปิดใหม่ดังนั้น กรุณาส่งกลับ หรือลองอีกครั้งหลังจากที่พอฝุ่น ฉันยังต้องรัฐ ว่าคนที่ มีทักษะของเราสามารถ ruin และเหลือเชื่อคิดเช่นการ อินเทอร์เน็ตสำหรับการสื่อสารมีการกระทบกระแทก ฉันได้พยายามหลายปี กลับมาเขียนให้ผู้อื่นสำหรับความสำเร็จของฉันทำงานขณะนี้ ปานกลาง ขอบคุณที่ให้ดอกเบี้ยทั้งหมดฉันได้รับ
เครื่องมือการตรวจสอบแบบฟอร์มนี้สามารถตรวจสอบ เขตข้อมูลจำนวนเต็ม อีเมล์ และวันที่ต้อง ในแบบฟอร์มของคุณ เป็นแบบพกพาได้อย่างง่ายดายในฟอร์มของคุณทั้งหมด และจะช่วยประหยัดฝั่งเซิร์ฟเวอร์ ทรัพยากรจากการตรวจสอบของเขตข้อมูลฟอร์ม และหน้าชขอบ ข้อความทั้งหมดได้รับการกำหนดเองในแบบฟอร์มที่ซ่อน องค์ประกอบที่คุณเลือกความต้องการตรวจสอบ คุณสามารถผสม องค์ประกอบจำเป็นต้อง มีองค์ประกอบอื่นใด ๆ รหัสนี้จะสมบูรณ์แบบพกพา และสามารถใช้ได้กับรูปแบบใดโดยเฉพาะ การเพิ่มองค์ประกอบที่ซ่อนอยู่ของคุณอย่างเหมาะสมและเพิ่มการ onClick ให้กับปุ่มคำสั่ง

ปฏิบัติตามกฎ
1) เมื่อต้องการตั้งค่าเขตข้อมูลที่ต้องใช้ชื่อของเขตข้อมูลที่มีอยู่ และเพิ่ม r_ การไปที่จุดเริ่มต้นของเขตข้อมูลชื่อ และสร้างองค์ประกอบที่ซ่อนอยู่
ตัวอย่าง:
<input type="text" name="your_fieldname">
<input type="hidden" name="r_your_fieldname" value="Required error message goes here">

2) การตั้งค่าเป็นเลขจำนวนเต็ม ฟิลด์ใช้ชื่อของเขตข้อมูลที่มีอยู่ และเพิ่ม i_ การไปที่จุดเริ่มต้นของเขตข้อมูลชื่อ และสร้างองค์ประกอบที่ซ่อนอยู่
ตัวอย่าง:
<input type="text" name="your_fieldname">
<input type="hidden" name="i_your_fieldname" value="Integer error message goes here">

3) เมื่อต้องการตั้งค่าวันเขตข้อมูลใช้ชื่อของเขตข้อมูลที่มีอยู่ และเพิ่มการ d_ ไปที่จุดเริ่มต้นของเขตข้อมูลชื่อ และสร้างองค์ประกอบที่ซ่อนอยู่ นี้ ฟิลด์วันยอมรับเฉพาะการจัดรูปแบบ MM/DD/YY หรือ d/M/YY
ตัวอย่าง:
<input type="text" name="your_fieldname">
<input type="hidden" name="d_your_fieldname" value="Date error message goes here">

4) เมื่อต้องการตรวจสอบ เขตข้อมูลอีเมล์ใช้ชื่อของเขตข้อมูลที่มีอยู่ และ เพิ่ม e_ เป็นจุดเริ่มต้นของชื่อเขตข้อมูล และสร้างเป็นที่ซ่อนอยู่ องค์ประกอบ
ตัวอย่าง:
<input type="text" name="your_fieldname">
<input type="hidden" name="e_your_fieldname" value="The email address format is incorrect">

เตรียมใช้งานสคริปต์
คุณต้องใช้ชนิด = "ปุ่ม" ไม่ชนิด = "ส่ง" สำหรับปุ่มส่งของคุณ javascript ส่งส่งให้คุณ
ตัวอย่าง:
<input type=button Value="Button Message Here" onclick="check(form,form.elements.length)">

ตัวอย่างนี้ใช้เครื่องมือตรวจสอบที่สามทั้งหมดในชุด

อีเมล์: (required and email validation)
ชื่อ: (required validation)
นามสกุล: (required validation)
ที่อยู่ 1: (required validation)
ที่อยู่ 2: (no validation)
คุณอายุเท่าไหร่: (required and integer validation)
ปีมีคุณรับสคริปต์: (integer validation)
วันของวันนี้คือ: (mm/dd/yy) (date validation)
เลือกหัวเรื่อง: (required validation)
เลือกปุ่มวิทยุ: ทดสอบ 1 ทดสอบ 2 ทดสอบ 3 (required validation)
เลือกทั้งหมดที่นำไปใช้: ทดสอบ 1 ทดสอบ 2 ทดสอบ 3 (required validation on box 1)
