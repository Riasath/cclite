자바 스크립트 양식 유효성 검사 도구 ver 2.5
Artswork에 의해
나는 사실에 대 한 사과 해야 더 이상 어디 것이 스크립트에 대 한 이메일 질문을 받아들일 수 제대로이 모듈을 통합 하는 사람들을 돕는에 많은 시간을 줄 고 누구를 위한 추가 항목을 작성 합니다. 이 이메일 이후 발생 했습니다. 2003 년 8 월의 웜 표시 내 이메일 주소를 심각 하 게 손상 여기 어디 수천, 잘못 대처 했다 공격 하루에 이메일을 반환 합니다. 정기적으로 다시 켜려면 시도해 볼 수도 있습니다 그래서 반환 하거나 먼지가 앉 았을 때 후 다시 시도 하십시오. 또한 상태 해야 합니다. 어떻게 우리의 능력을 가진 사람을 망칠 수 있습니다와 같은 놀라운 아이디어를 인터넷 통신에 대 한 비방. 오랜 세월 동안 시도 했다 나의 성공이 따라 작업에 대 한 다른 사람에 게 기부를 반환 매체입니다. 내가 받은 모든 관심에 감사 합니다.
이 양식 유효성 검사 도구에 대 한 확인할 수 있습니다. 필수, 정수, 이메일 및 날짜 필드 귀하의 형태로. 귀하의 모든 형태를 쉽게 휴대용 이며 서버 쪽 저장 도움이 될 것입니다. 양식 필드 유효성 검사에서 리소스 그리고 페이지를 다시 로드 합니다. 숨겨진된 폼에 있는 모든 메시지 사용자 지정 요소가 유효성 검사 요구 사항을 선택할 수도 있습니다. 혼합 수 있습니다. 다른 요소와 필수 요소입니다. 이 코드는 완전히 휴대용 하 고 그냥 어떤 양식을 사용할 수 있습니다. 숨겨진된 요소를 적절 하 게 추가 하 고 해당 onClick 추가 명령 단추입니다.

다음 규칙.
1) 필수 필드 사용 하 여 기존 필드 이름을 설정 하 고 추가 필드의 시작 부분에 r_ 이름과 숨겨진된 요소를 만듭니다.
예:
<input type="text" name="your_fieldname">
<input type="hidden" name="r_your_fieldname" value="Required error message goes here">

2) 정수 설정 하려면 필드 기존 필드 이름을 사용 하 고 추가 필드의 시작 부분에 i_ 이름과 숨겨진된 요소를 만듭니다.
예:
<input type="text" name="your_fieldname">
<input type="hidden" name="i_your_fieldname" value="Integer error message goes here">

3) 설정 날짜 필드 사용 하 여 기존 필드 이름을 추가 하는 필드의 시작 부분에 d_ 이름과 숨겨진된 요소를 만듭니다. 이 날짜 필드는 형식 MM/DD/YY 또는 YYYY-MM-DD만 허용합니다.
예:
<input type="text" name="your_fieldname">
<input type="hidden" name="d_your_fieldname" value="Date error message goes here">

4) 유효성을 검사 하려면 메일 필드 기존 필드 이름을 사용 하 고 한 e_ 필드 이름 앞에 추가 하 고 숨겨진 만들기 요소입니다.
예:
<input type="text" name="your_fieldname">
<input type="hidden" name="e_your_fieldname" value="The email address format is incorrect">

스크립트의 초기화 합니다.
Type을 사용 해야 합니다 "버튼" 하지 type = "submit" = 전송 단추에 대 한 javascript를 보냅니다 당신을 위해 제출.
예:
<input type=button Value="Button Message Here" onclick="check(form,form.elements.length)">
