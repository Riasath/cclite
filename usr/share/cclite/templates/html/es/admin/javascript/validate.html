Puede comprobar esta herramienta de validación de formulario campos obligatorios, entero, correo electrónico y fecha en los formularios. Es fácilmente portable a todos los formularios y ayudará a salvar del lado del servidor recursos de validación de campo de formulario y recarga de página. Todos los mensajes personalizados en forma oculta elemento cuando selecciona el requisito de validación. Se puede mezclar el elemento requerido con cualquiera de los otros elementos. Este código es completamente portátil y puede utilizarse con cualquier forma por sólo agregar los elementos ocultos adecuadamente y agregar el onClick comando a un botón.

Siguiendo las reglas.
1) Para definir un campo obligatorio uso el nombre de campo existentes y agregar una r_ al principio del campo nombre y cree un elemento oculto.
EJEMPLO:
<input type="text" name="your_fieldname">
<input type="hidden" name="r_your_fieldname" value="Required error message goes here">

2) Para definir un entero campo utilizar su nombre de campo existentes y agregar un i_ al principio del campo nombre y cree un elemento oculto.
EJEMPLO:
<input type="text" name="your_fieldname">
<input type="hidden" name="i_your_fieldname" value="Integer error message goes here">

3) Para establecer un uso de campo fecha su nombre de campo existentes y agregar un D_ al principio del campo nombre y cree un elemento oculto. Esto campo fecha sólo acepta el formato DD/MM/AA o M/D/AA.
EJEMPLO:
<input type="text" name="your_fieldname">
<input type="hidden" name="d_your_fieldname" value="Date error message goes here">

4) Para validar un campo de correo electrónico utilice su nombre de campo existentes y Agregar una e_ al principio del nombre del campo y crear un oculto elemento.
EJEMPLO:
<input type="text" name="your_fieldname">
<input type="hidden" name="e_your_fieldname" value="The email address format is incorrect">

Inicialización de la secuencia de comandos.
DEBE utilizar el tipo = "button" no el tipo = "Enviar" para el botón de envío, el código javascript envía el envío para usted.
EJEMPLO:
<input type=button Value="Button Message Here" onclick="check(form,form.elements.length)">
