Cclite Quickstart

Instalador Web | Inicio de sesión de usuario | Administración |
  1. Uso el análisis de la instalación para analizar el estado actual de la instalación
  2. Uso el instalador para configurar los valores de configuración, contraseña y usuario de base de datos
  3. Crear un registro (trading group) for your system, for example, Dalston , Newington , Islington Un registro es una base de datos de mysql...
  4. Inicio de sesión como administrador Inicio de sesión para su registro como Manager con contraseña Manager
  5. (when Manager) Set up a currency or two
  6. Crear usuarios o hacerles venir Crear usuarios y dejarlos a comercio.
  7. Recuerde quitar o deshabilitar al instalador y cambiar las contraseñas de administrador y sysaccount. Quitar esta página también!
Trabajos por lotes como escritura rss, leer entrada de transacción del lote y gráficas están ahora en la página de administración para el administrador. Que pueden ejecutar directamente desde la web sin utilizar cron. Para cron, unas modificaciones deban hacerse en las secuencias de comandos.
