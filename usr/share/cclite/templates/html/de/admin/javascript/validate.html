Dieses Formular-Validierung-Tool kann überprüfen Pflichtfelder, Integer, e-Mail und Datum in Ihren Formularen. Es ist leicht tragbar für alle Ihre Formen und hilft sparen Serverseite Ressourcen aus Feld Formularvalidierung und die Seite wird neu geladen. Alle Nachrichten werden in versteckten Form angepasst. Element, in dem Sie auch die Validierung Anforderung auswählen. Sie können mischen das erforderliche Element mit jedem der anderen Elemente. Dieser Code ist vollständig portabel und kann nur mit irgendeiner Form von verwendet werden die ausgeblendete Elemente entsprechend hinzufügen und onClick Befehl, um eine Schaltfläche.

Nach den Regeln.
1) Legen Sie einer Pflichtfeld Verwendung Ihrer vorhandenen Feldnamen und hinzufügen ein R_ bis zum Anfang des Felds name und erstellen ein ausgeblendetes Element.
BEISPIEL:
<input type="text" name="your_fieldname">
<input type="hidden" name="r_your_fieldname" value="Required error message goes here">

2) Um eine Ganzzahl Feld verwenden Sie Ihre vorhandenen Feldnamen und fügen ein I_ an den Anfang des Feldes name und erstellen ein ausgeblendetes Element.
BEISPIEL:
<input type="text" name="your_fieldname">
<input type="hidden" name="i_your_fieldname" value="Integer error message goes here">

3) Legen Sie ein Datum-Feld verwenden Ihre vorhandenen Feldnamen und Hinzufügen einer D_ bis zum Anfang des Felds name und erstellen ein ausgeblendetes Element. Dies Datumsfeld akzeptiert nur das Format MM/TT/JJ oder m/D/YY.
BEISPIEL:
<input type="text" name="your_fieldname">
<input type="hidden" name="d_your_fieldname" value="Date error message goes here">

4) Ein e-Mail-Feld verwenden überprüfen Ihre vorhandene Feldnamen und Beginn des Feldnamens eine E_ hinzu und erstellen Sie eine versteckte Element.
BEISPIEL:
<input type="text" name="your_fieldname">
<input type="hidden" name="e_your_fieldname" value="The email address format is incorrect">

Die Initialisierung des Skripts.
MÜSSEN Sie den Typ = "button" nicht den Typ = "submit" für die Schaltfläche Absenden sendet das Javascript senden für Sie.
BEISPIEL:
<input type=button Value="Button Message Here" onclick="check(form,form.elements.length)">
