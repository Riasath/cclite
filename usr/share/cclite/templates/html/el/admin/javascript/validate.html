Αυτό το εργαλείο επικύρωσης φόρμας μπορεί να ελέγξει για απαιτείται, integer, ηλεκτρονικό ταχυδρομείο και ημερομηνία πεδία στις φόρμες σας. Είναι εύκολα φορητό σε όλες τις φόρμες σας και θα εξοικονομήσετε διακομιστή μέσα από την επικύρωση πεδίου φόρμας και φορτώνει η σελίδα. Όλα τα μηνύματα που έχουν προσαρμοστεί σε μια κρυφή φόρμα στοιχείο όπου επιλέγετε επίσης την απαίτηση επικύρωσης. Μπορείτε να αναμείξετε το απαιτούμενο στοιχείο με οποιοδήποτε από τα άλλα στοιχεία. Αυτός ο κωδικός είναι εντελώς φορητή και μπορεί να χρησιμοποιηθεί με οποιαδήποτε φόρμα από μόλις Προσθήκη σας κρυφά στοιχεία κατάλληλα και προσθέτοντας το onClick η εντολή σε ένα κουμπί.

Σύμφωνα με τους κανόνες.
1) Για να ορίσετε ένα απαιτούμενο πεδίο χρήση σας όνομα υπάρχοντος πεδίου και προσθέσετε μια r_ στην αρχή του πεδίου όνομα και δημιουργήστε ένα κρυφό στοιχείο.
ΠΑΡΆΔΕΙΓΜΑ:
<input type="text" name="your_fieldname">
<input type="hidden" name="r_your_fieldname" value="Required error message goes here">

2) Για να ορίσετε έναν ακέραιο πεδίο χρησιμοποιήστε σας όνομα υπάρχοντος πεδίου και προσθέστε μια i_ στην αρχή του πεδίου όνομα και δημιουργήστε ένα κρυφό στοιχείο.
ΠΑΡΆΔΕΙΓΜΑ:
<input type="text" name="your_fieldname">
<input type="hidden" name="i_your_fieldname" value="Integer error message goes here">

3) Για να ορίσετε μια ημερομηνία το πεδίο χρήση σας όνομα υπάρχοντος πεδίου και προσθέσετε ένα d_ στην αρχή του πεδίου όνομα και δημιουργήστε ένα κρυφό στοιχείο. Αυτό πεδίο "ημερομηνία" αποδέχεται μόνο το μορφή μμ/ηη/ΕΕ ή ηη/Μ/ΕΕ.
ΠΑΡΆΔΕΙΓΜΑ:
<input type="text" name="your_fieldname">
<input type="hidden" name="d_your_fieldname" value="Date error message goes here">

4) Για να επικυρώσετε ένα πεδίο ηλεκτρονικού ταχυδρομείου χρησιμοποιούν σας όνομα υπάρχοντος πεδίου και προσθέσετε ένα e_ στην αρχή του ονόματος του πεδίου και να δημιουργήσετε ένα κρυφό το στοιχείο.
ΠΑΡΆΔΕΙΓΜΑ:
<input type="text" name="your_fieldname">
<input type="hidden" name="e_your_fieldname" value="The email address format is incorrect">

Προετοιμασία της δέσμης ενεργειών.
Πρέπει να χρησιμοποιήσετε τον τύπο = "κουμπί" δεν τον τύπο = "Υποβολή" για το κουμπί "ΥΠΟΒΟΛΉ", το javascript στέλνει την υποβολή σας.
ΠΑΡΆΔΕΙΓΜΑ:
