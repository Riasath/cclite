Этот инструмент проверки формы можно проверить наличие целое число, электронной почты и дата поля в формах. Это легко совместимо для всех ваших форм и поможет спасти стороне сервера ресурсы от проверки поля формы и Страница перезагрузится. Все сообщения будут настроены в скрытой форме элемент, где также выбрать требование проверки. Вы можете смешивать обязательный элемент с любым из других элементов. Этот код является полностью переносимым и может использоваться с любой формой, просто Добавление ваши скрытые элементы соответствующим образом и добавление onClick команда на кнопку.

По правилам.
1) Чтобы задать обязательное поле использовать существующее имя поля и добавить r_ до начала поля имя и создать скрытый элемент.
ПРИМЕР:
<input type="text" name="your_fieldname">
<input type="hidden" name="r_your_fieldname" value="Required error message goes here">

2) Чтобы задать целое поле использовать существующее имя поля и добавить i_ до начала поля имя и создать скрытый элемент.
ПРИМЕР:
<input type="text" name="your_fieldname">
<input type="hidden" name="i_your_fieldname" value="Integer error message goes here">

3) Чтобы задать использование поля Дата существующее имя поля и добавить D_ в начало поля имя и создать скрытый элемент. Это поле «Дата» принимает только в формате мм/дд/гг или ДД/М/гг.
ПРИМЕР:
<input type="text" name="your_fieldname">
<input type="hidden" name="d_your_fieldname" value="Date error message goes here">

4) Чтобы проверить поле электронной почты использовать существующее имя поля и Добавьте в начало имени поля e_ и создайте скрытый элемент.
ПРИМЕР:
<input type="text" name="your_fieldname">
<input type="hidden" name="e_your_fieldname" value="The email address format is incorrect">

Инициализация сценария.
Вы должны использовать тип = «кнопка» не type = «отправить» для вашей кнопки отправки, javascript посылает submit для вас.
ПРИМЕР:
<input type=button Value="Button Message Here" onclick="check(form,form.elements.length)">

Этот пример использует все средства три проверки в комбинации.
