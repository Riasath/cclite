Programme d'installation Web | Ouverture de session utilisateur | Administration |
  1. Utilisation l'analyse de l'installation pour analyser l'état actuel de l'installation
  2. Utilisation le programme d'installation pour configurer les valeurs de configuration, mot de passe et utilisateur de la base de données
  3. Créer un registre (trading group) for your system, for example, Dalston , Newington , Islington Un registre est une base de données mysql...
  4. Ouverture de session comme manager Connexion à votre registre : Gestionnaire avec mot de passe Gestionnaire
  5. (when Manager) Set up a currency or two
  6. Créez des utilisateurs ou de les laisser venir Créer des utilisateurs et de laisser le commerce.
  7. N'oubliez pas de supprimer ou de désactiver le programme d'installation et de modifier le mot de passe sysaccount et admin. Supprimer cette page aussi !
Lots comme l'écriture de flux rss, lecture lot transaction d'entrée et les graphiques sont maintenant sur la page de gestion de l'administrateur. Ils peuvent être exécuté directement à partir du web sans utiliser cron. Pour cron, quelques modifications doivent être faites dans les scripts.
