Cet outil de validation de formulaire peut vérifier champs obligatoires, en entier, email et date dans vos formulaires. Il est facilement portable dans tous vos formulaires et contribuera à sauver côté serveur ressources de validation de champ de formulaire et la recharge de la page. Tous les messages sont personnalisés sous forme cachée Lorsque vous sélectionnez également la condition de validation de l'élément. Vous pouvez mélanger l'élément requis avec aucun des autres éléments. Ce code est entièrement portable et peut être utilisé avec toute forme de juste ajouter vos éléments cachés convenablement et en ajoutant l'onClick commande à un bouton.

Suivant les règles.
1) Pour définir une utilisation du champ obligatoire votre nom de domaine existant et ajoutez un r_ au début du champ nom et créer un élément caché.
EXEMPLE :
<input type="text" name="your_fieldname">
<input type="hidden" name="r_your_fieldname" value="Required error message goes here">

2) Pour définir un nombre entier champ utiliser votre nom de domaine existant et ajoutez un i_ au début du champ nom et créer un élément caché.
EXEMPLE :
<input type="text" name="your_fieldname">
<input type="hidden" name="i_your_fieldname" value="Integer error message goes here">

3) Pour définir une date champ utiliser votre nom de domaine existant et ajouter un d_ au début du champ nom et créer un élément caché. Ce champ de date n'accepte que le format MM/JJ/AA ou M/YY.
EXEMPLE :
<input type="text" name="your_fieldname">
<input type="hidden" name="d_your_fieldname" value="Date error message goes here">

4) Pour valider un champ email utilisez votre nom de domaine existant et ajouter un f_ au début du nom du champ et créer un caché élément.
EXEMPLE :
<input type="text" name="your_fieldname">
<input type="hidden" name="e_your_fieldname" value="The email address format is incorrect">

Initialisation du script.
Vous devez utiliser le type = « bouton » pas le type = « submit » pour votre bouton SUBMIT, le javascript envoie le soumettre pour vous.
EXEMPLE :
<input type=button Value="Button Message Here" onclick="check(form,form.elements.length)">
