Cclite Quickstart

Instalador da Web | Login do usuário | Administração |
  1. Utilização a análise de instalação para analisar o estado atual da instalação
  2. Utilização o instalador para definir os valores de configuração, senha e usuário de banco de dados
  3. Criar um registro (trading group) for your system, for example, Dalston , Newington , Islington Um registro é um banco de dados mysql...
  4. Faça logon como Gerenciador de Início de sessão para seu registro como Gerenciador com senha Gerenciador
  5. (when Manager) Set up a currency or two
  6. Criar usuários ou deixá-los chegar Criar usuários e deixá-los de comércio.
  7. Lembre-se de remover ou desabilitar o instalador e alterar as senhas de administrador e sysaccount. Remova esta página também!
Trabalhos em lotes como gravar rss, ler entrada de transação em lote e gráficos estão agora na página de gerenciamento para o administrador. Eles podem ser executado diretamente da web sem usar o cron. Para o cron, algumas modificações precisam ser feitas nos scripts.
