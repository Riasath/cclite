Esta ferramenta de validação de formulário pode verificar campos obrigatórios, inteiro, e-mail e data em seus formulários. É facilmente portável para todas as suas formas e ajudará a salvar do lado do servidor recursos de validação de campo de formulário e recarrega a página. Todas as mensagens são personalizadas no formulário oculto elemento onde você seleciona também o requisito de validação. Você pode misturar o elemento exigido com qualquer um dos outros elementos. Este código é totalmente portátil e pode ser usado com qualquer forma por apenas adicionando seus elementos ocultos apropriadamente e adicionando o onClick comando para um botão.

Seguindo as regras.
1) Para definir um campo necessário usar seu nome de domínio existente e adicionar um r _ até o início do campo nome e criar um elemento oculto.
EXEMPLO:
<input type="text" name="your_fieldname">
<input type="hidden" name="r_your_fieldname" value="Required error message goes here">

2) Para definir um número inteiro campo usar seu nome de domínio existente e adicionar um i _ para o início do campo nome e criar um elemento oculto.
EXEMPLO:
<input type="text" name="your_fieldname">
<input type="hidden" name="i_your_fieldname" value="Integer error message goes here">

3) Definir um uso de campo de data, seu nome de domínio existente e adicionar um d_ para o início do campo nome e criar um elemento oculto. Isso campo de data só aceita o formato DD/MM/AA ou dd/mm/YY.
EXEMPLO:
<input type="text" name="your_fieldname">
<input type="hidden" name="d_your_fieldname" value="Date error message goes here">

4) Para validar um campo de email use seu nome de domínio existente e Adicione um e _ no início do nome do campo e criar um oculto elemento.
EXEMPLO:
<input type="text" name="your_fieldname">
<input type="hidden" name="e_your_fieldname" value="The email address format is incorrect">

Inicialização do script.
Você deve usar o tipo = "button" não o tipo = "submit" para o botão enviar, o javascript envia o envio para você.
EXEMPLO:
<input type=button Value="Button Message Here" onclick="check(form,form.elements.length)">

Este exemplo utiliza todas as ferramentas de validação três em combinação.

Endereço de e-mail: (required and email validation)
Primeiro nome: (required validation)
Sobrenome: (required validation)
Endereço 1: (required validation)
Endereço 2: (no validation)
Qual é a sua idade: (required and integer validation)
Anos você tem sido scripts: (integer validation)
O que é hoje: (mm/dd/yy) (date validation)
Selecione um assunto: (required validation)
Selecione um botão de opção: ensaio 1 ensaio 2 teste 3 (required validation)
Selecione todos os que se aplicam: ensaio 1 ensaio 2 teste 3 (required validation on box 1)
